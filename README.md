## Cài đặt và chạy dự án

1. Clone dự án
- git clone https://github.com/Tducdz/Gigaripi-PC.git
- cd Gigaripi-PC
2. Cài đặt dependencies
- npm install
3. Chạy tailwind (watch mode)
- npm run tailwind:watch
4. Mở index.html

## Công nghệ sử dụng
- HTML5
- CSS3 / TailwindCSS v4 
- JavaScript (ES6+)
- Node.js + npm
